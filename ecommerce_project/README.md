# Ecommerce Store

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Features
- User Registration
- User Login
- Product Listing
- Product Details
- Shopping Cart
- Wishlist
- Quantity Increase/Decrease
- Checkout
- Order Processing
- Admin Dashboard
- Product Management

## Installation

```bash
git clone <repository-url>
cd ecommerce-project

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
